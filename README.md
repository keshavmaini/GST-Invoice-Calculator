# GST & Invoice Calculator

A modern and responsive GST & Invoice Calculator built using HTML, CSS, and JavaScript. The application helps users calculate GST amounts, total invoice values, and tax breakdowns quickly and accurately.

## Features

* GST calculation in real-time
* Total invoice amount calculation
* CGST, SGST, and IGST breakdown
* Standard GST rates (5%, 12%, 18%, 28%)
* Custom GST rate support
* Dark and Light mode toggle
* Copy results to clipboard
* Input validation and error handling
* Responsive design for all devices
* Modern glassmorphism user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Getting Started

### Run Locally

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in your browser.

Alternatively, use the Live Server extension in VS Code for development.

## Deployment

This project can be deployed on any static hosting platform, including:

* Vercel
* GitHub Pages
* Netlify

Simply upload the project files and deploy.
